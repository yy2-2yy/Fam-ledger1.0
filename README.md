# 家庭周记账

一个本地优先的手机网页 PWA，用来按周整理微信/支付宝消费记录、维护自定义分类，并查看每日消费趋势。

## 本地运行

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

然后打开：

```text
http://localhost:4173
```

## 功能

- 手动新增消费记录：日期、金额、来源、商户 / 备注、分类。
- 粘贴账单文本并解析为待确认记录。
- 自定义分类：新增、改名、改颜色、隐藏。
- 规则记忆：同一商户下次自动推荐分类。
- 折线图：按周/月查看每日总消费和多选分类消费变化。
- 本地 IndexedDB 存储，不需要账号或云端。
- 导出 JSON 完整备份、导出 CSV 记录、导入 JSON 恢复。

## 备份提醒

数据只保存在当前浏览器本地。换手机、清浏览器数据或卸载浏览器前，请先在“备份”页导出完整备份。
